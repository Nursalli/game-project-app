# Game Project App

## Install
```sh
$ yarn 
# or
$ npm install
```

## Setup

Set up your `.env` file align with `.env.sample` given

```.env
VITE_BACKEND_BASE_URL=https://testing-game-project-service.herokuapp.com/
```
## Run

```sh
$ yarn dev
```
